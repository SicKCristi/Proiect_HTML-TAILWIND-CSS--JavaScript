PROIECTUL 3
CANVAS IMPLEMENTAREA TEMEI BIBLIOGRAFIE

Salut tuturor! Astăzi, în cadrul acestui proiect voi încerca să folosesc noile concepte învățate despre limbajul de programare JavaScript de la disciplina Tehnologii Web pentru a crea un canvas care face mirror unei imagini generate de un API, pe site.

API-ul NASA este o resursă de date extraordinară, furnizată de Administrația Națională Aeronautică și Spațială a Statelor Unite. Acest API oferă acces la o varietate amplă de informații și date spațiale. Este utilizat pe scară largă în domeniul științei, educației și explorării spațiului. În cazul nostru, vom genera o Imagine cu un cățeluș, pe care o vom oglindi și vom face o temă de procesare pe aceasta(eu am ales tema de transformare în nuanțe de gri a unei imagini color sau Tema de procesare 1 din fisierul cu tema)

Construirea acestui site a fost o provocare incitantă, marcând explorarea JavaScript-ului și prima experiență practică cu Tailwind CSS. În centrul proiectului a stat învățarea profundă a interacțiunii dintre JavaScript și structura unui site web, Tailwind CSS oferind un cadru flexibil și eficient pentru stilizare și responsivitate. Procesul a reprezentat nu doar o abordare practică a dezvoltării web, ci și o incursiune captivantă în aspectele fundamentale ale designului și funcționalității.
